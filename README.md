1. **Clone the repository**

2. **Navigate to the project directory**
   ```bash
   cd nba-draft-hub
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
